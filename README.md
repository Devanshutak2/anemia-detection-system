# 🩸 Anemia Detection System

## 📌 Overview
The Anemia Detection System is a web-based application developed using Django that helps in identifying anemia through image-based analysis of body parts such as eyes, nails, or palms.

This system aims to provide a quick and accessible preliminary diagnosis using digital image processing techniques.

---

## 🚀 Features
- 📷 Upload images for anemia detection
- ⚡ Real-time analysis
- 🧠 Intelligent detection logic
- 🌐 User-friendly web interface
- 📊 Fast and efficient results

---

## 🛠️ Tech Stack
- Backend: Python (Django)
- Frontend: HTML, CSS, JavaScript
- Database: SQLite
- Tools: VS Code, Git, GitHub
